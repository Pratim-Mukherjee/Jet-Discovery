# Hybrid Physics Ensemble Autoencoder

Project structure for a physics-informed hybrid autoencoder ensemble.

## Structure

- `data/`
- `models/`
- `physics/`
- `training/`
- `api/`
- `scripts/`
- `config.yaml`
- `requirements.txt`

## Setup

1. Create a virtual environment
2. Install dependencies: `pip install -r requirements.txt`
